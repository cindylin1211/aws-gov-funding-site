# AWS 政府補助網站維護指南

## 📋 概述
這份指南將幫助 Maggie 獨立維護和更新 AWS 政府補助網站，無需技術背景即可新增補助計畫。

## 🚀 快速新增補助計畫

### 方法 1: 直接編輯 JSON 文件（推薦）

1. **找到資料文件**
   - 打開 `public/grants-database.json` 文件
   - 這個文件包含所有補助計畫資料

2. **複製現有計畫格式**
   ```json
   {
     "id": "new-grant-id",
     "計畫名稱": "新計畫名稱",
     "補助類別": "數位轉型", // 或 "創新研發", "人才培訓"
     "子分類": "AI智慧化",
     "補助重點": "計畫重點說明",
     "補助對象": [
       "申請條件1",
       "申請條件2"
     ],
     "補助金額": "最高金額",
     "補助比例上限": "50%",
     "計畫時程": "申請和執行時間",
     "主辦單位": "主辦機關",
     "參考資料": [
       {
         "text": "計畫官網",
         "url": "https://example.com"
       }
     ],
     "企業規模": ["中小企業", "大型企業"],
     "金額分類": "中額補助", // 小額/中額/大額/超大額/無上限補助
     "主辦機關分類": "經濟部"
   }
   ```

3. **新增到 grants 陣列中**
   - 將新計畫加到 `"grants": [` 陣列的最前面
   - 確保 JSON 格式正確（注意逗號和括號）

### 方法 2: 使用管理介面（未來開發）

目前網站有 `/admin` 頁面框架，可以擴展為圖形化管理介面。

## 🏷️ 標籤分類指南

### 補助類別（大標籤）
- **數位轉型**: AI、智慧化、數位化相關
- **創新研發**: 技術開發、產品研發
- **人才培訓**: 教育訓練、技能提升

### 子分類（小標籤）
- AI智慧化、服務業創新、傳統產業升級
- 半導體技術、雲端服務、前瞻技術
- 大型企業培訓、數位轉型培訓等

### 金額分類
- **小額補助**: 100萬以下
- **中額補助**: 100-500萬
- **大額補助**: 500-1000萬
- **超大額補助**: 1000萬以上
- **無上限補助**: 無金額上限

## 🔧 網站部署流程

### 使用 GitHub（推薦）
1. 登入 GitHub: https://github.com/cindylin1211/aws-gov-funding-site
2. 編輯 `public/grants-database.json` 文件
3. 提交更改（Commit changes）
4. Amplify 會自動部署更新

### 手動部署
1. 下載整個專案
2. 修改 `public/grants-database.json`
3. 上傳到 AWS Amplify Console

## 📚 常見問題

### Q: 如何修改網站文字？
A: 編輯 `src/pages/Index.tsx` 文件中的文字內容

### Q: 如何更新聯絡資訊？
A: 在 `src/pages/Index.tsx` 文件中找到 footer 區域並修改

### Q: 網站無法訪問怎麼辦？
A: 檢查 AWS Amplify Console 的部署狀態，或聯絡技術支援

### Q: 如何備份資料？
A: 定期下載 `public/grants-database.json` 文件作為備份

## 🎯 最佳實踐

1. **定期備份**: 每次更新前先備份 JSON 文件
2. **測試更改**: 在本地測試或使用預覽功能
